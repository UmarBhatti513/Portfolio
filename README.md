# Muhammad Umar Bhatti - Portfolio Website

## Introduction
This is a personal portfolio website for Muhammad Umar Bhatti, showcasing his skills, projects, experience, and certifications. The site is built with HTML, CSS, and JavaScript, featuring a responsive design, dark mode toggle, smooth animations, and interactive elements.

## Features
- **Responsive Design**: Works seamlessly on desktops, tablets, and mobile devices.
- **Dark/Light Mode Toggle**: User preference is saved in local storage.
- **Smooth Scrolling & Animations**: Elements fade in as you scroll, and interactive hover effects are applied to cards and buttons.
- **Typing Animation**: The hero title and subtitle are animated with a typewriter effect.
- **Project Showcase**: Three featured projects with descriptions and technology tags.
- **Experience Timeline**: A visual timeline of work, internships, and education.
- **Certifications Grid**: Displays recent certifications with icons and dates.
- **Contact Section**: Links to email, phone, and LinkedIn (placeholder).
